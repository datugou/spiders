### 获取验证码答案思路
验证码的问题类似“写出每张图片有几只猫”，“标记每张图片是不是有雪”等等。
测试发现验证问题、以及图片都是固定的几个；每次的图片链接的是随机的生成的。
所以思路就是把所有验证问题和图片都获取，然后为每个图片做好标记，存储在本地。
后续验证时，下载当前的图片，然后与本地存的图片比较，就能得到验证码的答案。

`captcha_img_data` 文件为所有验证码图片的标记数据，经过算法变成了 hash 值，方便图片对比。
