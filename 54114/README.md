爬取全站所有企业，包含行业分类信息，省市信息，公司基本信息，如下：
```python
data = {
       '公司名称': {'省份': '',
                   '城市': '',
                   '一级行业': '',
                   '二级行业': '',
                   '链接': '',
                   '法定代表人': '',
                   '电话': '',
                   '邮箱': '',
                   '网址': '',
                   '地址': '',
                   '经营范围': '',
                   '公司简介': ''}
       }
```  

其他没什么说明的，网站规则比较简单，也没有反爬措施。

不过我们作为良好网民，不要对目标服务器造成过大的访问压力，最好限速爬取。
